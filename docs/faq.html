Chapter 3. FAQ

    Frequently asked questions

    3.1 Where is the DarGUI configuration file?

    • DarGUI stores user data in a file named dargui.rc. If the folder .config already exists in your HOME directory when DarGUI is first run then you will find dargui.rc in .config/dargui in your home directory, otherwise it will be in a folder named .dargui in your home directory.
