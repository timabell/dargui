Chapter 2. Using Dargui

Restoring archives

Individual files can be restored from the archive by selecting them, right-clicking with the mouse on the archive contents, and selecting "Restore selected".

You can select multiple files for restoration this way by holding down the Ctrl key while left-clicking with the mouse. Groups of files or directories can be selected by holding down the Alt key while left-clicking with the mouse.

If an unexpanded directory is selected (ie: the directory contents are hidden) then all its contents will also be selected. However, if its contents are visible then only the directory itself will be selected, allowing you to choose which of its contents you want to restore.

Files can also be selected using filters. To do this choose the "Select by Filter" option from the popup menu or the "Archive" menu.
