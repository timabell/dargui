Chapter 1. Introduction

Limitations of DarGUI

- non-English versions of Dar: users of systems which are set up to use the French, German or Swedish languages may find that they have a version of Dar which uses the appropriate language for the system. At present (v. 0.5.0) many of DarGUI's functions will not work on these systems. This problem will be resolved in a future release of DarGUI.

- Unimplemented features of Dar: While DarGUI attempts to provide access to the most common functionality of Dar, there are many other features which are not implemented. These include:

  • Extended Attributes
  • include-from-file/exclude-from-file (-[ and -] ) options
  • Merging of archives
  • Execution of a script or command between slices
  • Support for the nodump option
