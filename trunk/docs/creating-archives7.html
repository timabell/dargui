Chapter 2. Using DarGUI

Creating archives: Saved settings

The settings in the "New archive" tabs can be saved and reloaded using the buttons at the bottom left of the window.

Save settings - you will be asked to choose a location to save the file and a suitable filename. Settings files are given the suffix ".dargui" by default. You can also enter some notes about the backup if you want.
A settings file can be used to define default settings for creating archives, using the User preferences dialog.

Load settings - this enables you to reload settings from a saved file.
